# Atsuko Frontend

The official landing page for the [**Atsuko**](https://discord.com/oauth2/authorize?client_id=1130772002580475954&permissions=8&integration_type=0&scope=bot) Discord bot

## Tech Stack

This project is built with:
- **Next.js** for the framework and routing
- **Tailwind CSS** for styling and UI components
- **Framer Motion** for animations and transitions
- **Lucide React** for icons

## Getting Started

If you want to run this locally or explore the code, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/TunxxVN/atsuko-frontend.git
   cd atsuko-frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
